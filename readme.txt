Om te starten heb ik gekozen voor een game classe die verantwoordelijk is voor alles van het spel en het inlaad. Alle gameobjecten zijn opgebouwd uit objecten omdat "we prefer composition over inheritance". Verder gebruik ik ook het factory pattern voor het gemakelijk creëren van enemies. Het level bestaat ui gameobjecten met tile components en gebruikt een hexagonaal coördinaat systeem (wat ik misschien beter niet had gedaan, want het is nogal onïntuitief). Voor de uitleg van de controlls en het switchen van gamemode gebruik ik de imgui. Je kan spelen met toetsenbord en controller waarvoor ik xinput gebruik. Voor het geluid gebruik is de Simple SDL2 Audio die op een aparte thread loopt. De health en points displays gebruiken het observer patern. En de input gebruikt het command pattern.
